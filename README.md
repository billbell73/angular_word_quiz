angular_word_quiz
=================

##To run app locally:

```
npm install
bower install
```

####Unit tests:
```
karma start 
```

####e2e tests:
```
node app.js  // please keep node server running in separate terminal window
protractor protractor.conf.js
```   



####App itself:

```
node app.js
```
then point browser to [http://localhost:3000/game](http://localhost:3000/game)
