angular_word_quiz
=================

##To run locally:

```
npm install
bower install
```

####For unit tests:
```
karma start 
```

####For e2e tests:
```
node app.js  //*please keep node server running in separate terminal window, then:*
protractor protractor.conf.js

```

####To run app itself

```
node app.js
```
then point browser to http://localhost:3000/game
